## Executive Summary

The CCIDC Application Form is a website that features information about the 4 pathways to CID certification. It includes an FAQ about certification overview and what users need to apply. There is a prescreening questionnaire that that directs users to the appropriate pathway, and users can apply to different pathways through an online application form.

For more information on the technical aspects of the project, please visit [https://github.com/TritonSE/CCIDC-Application-Overhaul](https://github.com/TritonSE/CCIDC-Application-Overhaul).
